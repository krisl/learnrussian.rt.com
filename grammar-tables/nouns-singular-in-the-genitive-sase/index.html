Nouns Singular in the Genitive Сase

Print version (PDF 911 KB)
Lesson 14

The Genitive Case answers the questions Кого? Чего? and can be used to express
the absence of somebody or something:

У меня есть собака. (I have a dog) — У тебя нет собаки. (You don’t have a
dog.)

нет + object in Genitive

The Genitive Case is used to express the absence of any ingredient. If you wish to order
a pizza without cheese (for example), you should use the preposition без (without)
and an object in the genitive:

Я хочу пиццу без сыра. И кофе без молока.

без нет не было не будет

Genitive endings vary according to noun’s gender

она

  • У меня есть + что?
    (Nominative case)
  • У меня есть виза.
  • У тебя есть лампа.
  • У него есть машина.
  • У неё есть работа.
  • У нас есть семья.
  • У вас есть идея.
  • У них есть проблема.
  • У нас есть дверь.
  • У меня нет + чего?
    (Genitive case)
  • У меня нет визы.
  • У тебя нет лампы.
  • У него нет машины.
  • У неё нет работы.
  • У нас нет семьи.
  • У вас нет идеи.
  • У них нет проблемы.
  • У нас нет двери.

Ending: -а → -ы, -я → -и, -ь → -и

он/оно

  • У меня есть друг.
  • У тебя есть брат.
  • У него есть паспорт.
  • У неё есть муж.
  • У вас есть конкурент.
  • У них есть офис.
  • У нас есть календарь.
  • У нас есть пиво.
  • У нас есть море.
  • У меня нет друга.
  • У тебя нет брата.
  • У него нет паспорта.
  • У неё нет мужа.
  • У вас нет конкурента.
  • У них нет офиса.
  • У нас нет календаря.
  • У нас нет пива.
  • У нас нет моря.

Ending: zero ending → -а, -ь → -я, -о → -а, -е → -я

Remember

У меня есть время. — У меня нет времени.

У меня есть деньги. — У меня нет денег.

У меня есть мать. — У меня нет матери.

У меня есть дочь. — У меня нет дочери.

Genitive and Tenses

Вчера (Past Tense)

Сегодня (Present Tense)

Завтра (Future Tense)

Positive construction

  • У меня был дом.
  • У меня была квартира.
  • У меня было вино.
  • У меня были деньги.
  • У меня есть дом.
  • У меня есть квартира.
  • У меня есть вино.
  • У меня есть деньги.
  • У меня будет дом.
  • У меня будет квартира.
  • У меня будет вино.
  • У меня будут деньги.

Negative construction

  • У меня не было дома.
  • У меня не было квартиры.
  • У меня не было вина.
  • У меня не было денег.
  • У меня нет дома.
  • У меня нет квартиры.
  • У меня нет вина.
  • У меня нет денег.
  • У меня не будет дома.
  • У меня не будет квартиры.
  • У меня не будет вина.
  • У меня не будет денег.

Remember

У меня никогда не было квартиры. (double negation + Genitive)

У меня всегда была квартира.
