Expressing prohibitions. Imperative Mood of Russian verbs in negative constructions, the word “нельзя”.

Не курите тут! Тут нельзя курить!

In this lesson you will continue studying the Imperative Mood, learn how to use it in negative constructions and with the word нельзя.

For example:

Не курите здесь!

Здесь нельзя курить!

Задание 3. Перефразируйте предложения. Замените словосочетание «нельзя + глагол» на императив

Task 3. Look at the example and rephrase the sentences using “не” + Imperative “вы”-form. Use the on-screen keyboard.

Example: Тут нельзя курить! — Не курите тут!

  1. Нельзя пить много! – не пейте много!
  2. Нельзя ходить по газонам! — не ходите по газонам!
  3. Нельзя пить воду из реки! — не пейте воду из реки!
  4. Здесь нельзя парковать машины! — не паркуйте машины здесь!
  5. Нельзя слушать громкую музыку после одиннадцати часов вечера. — не слушайте громкую музыку после одиннадцати часов вечера.
  6. Нельзя опаздывать на встречу. — не опаздывайте на встречу.
  7. Нельзя забывать о родителях. — не забывайте о родителях.
  8. Нельзя пить алкоголь рядом со школой. — не пейте алкоголь рядом со школой.
  9. Нельзя долго спать. — не спите долго.
  10. Нельзя кидать мусор на улице. — не кидайте мусор на улице.
